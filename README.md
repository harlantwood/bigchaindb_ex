# BigchaindbEx

A driver for bigchaindb written in elixir

## Make sure you have the following installed:
  * elixir >= v1.5
  * libsodium >= v1.0.12

## Installation

BigchaindbEx can be installed
by adding `bigchaindb_ex` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:bigchaindb_ex, "~> 0.1"}
  ]
end
```
